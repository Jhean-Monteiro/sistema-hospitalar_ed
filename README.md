## 🚀 Funcionalidades

O sistema permite o gerenciamento de três setores distintos:
1. **Emergência (Pilha):** Atendimento baseado no princípio LIFO (*Last-In, First-Out*), onde casos urgentes são priorizados.
2. **Consultas (Fila Linear):** Atendimento por ordem de chegada convencional.
3. **Exames (Fila Circular):** Gerenciamento eficiente para exames, otimizando o uso do espaço de memória.

## 🛠️ Tecnologias e Conceitos Utilizados

- **Linguagem:** C
- **Estruturas de Dados:** - Pilha (Stack)
  - Fila (Queue)
  - Fila Circular (Circular Queue)
- **Modularização:** Divisão do código em arquivos `.h` e `.c` para melhor manutenção e organização.
